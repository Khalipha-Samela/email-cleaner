# Email Cleaner

A modern email management platform designed to help users clean, organize, and manage their inboxes efficiently.

## Current Features

### Frontend

* React.js
* Tailwind CSS
* React Router
* Responsive dashboard
* Modern SaaS-inspired UI
* Login screen
* Dashboard navigation
* Search interface
* Logout functionality

### Backend

* Node.js
* Express
* REST API foundation
* Health monitoring endpoint
* CORS support

## Dashboard

Current dashboard includes:

* Inbox overview
* Statistics cards
* Cleanup opportunities table
* Gmail connection placeholder
* Sidebar navigation
* User profile section

## Technology Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Axios
* Lucide React
* React Router DOM

### Backend

* Node.js
* Express
* CORS
* Dotenv
